# esx_accessories

Boutiques d'accessoires (chapeau/casque, lunettes, masques, accessoires d'oreilles). Vous pouvez mettre ou enlever les accessoires avec un menu. Accessoires enregistrés dans la base de données.

/!\ Fonctionne uniquement avec les accessoires achetés dans les zones spéciales du magasin de vêtements, pas via le script esx_clotheshop.


## REQUIREMENTS

- [esx_skin](https://github.com/ESX-Org/esx_skin)
- [esx_datastore](https://github.com/ESX-Org/esx_datastore)

## INSTALLATION

1. Ajoutez a votre resources/[esx] folder
2. Importez esx_accessories.sql dans votre base de donnée
3. Ajoutez a votre server.cfg :

```

> ensure esx_accessories

```
